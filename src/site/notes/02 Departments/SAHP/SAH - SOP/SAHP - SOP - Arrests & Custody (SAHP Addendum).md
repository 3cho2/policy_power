# SAHP - SOP - Arrests & Custody (SAHP Addendum)

## 1) Alignment statement
Shared custody/booking policy is authoritative.

## 2) Booking location guidance
- Each agency has its own jail.
- Any jail may be used when needed.
- Advised: SAHP primary/arresting unit uses SAHP booking when practical.

## 3) Non-arresting transport requirement
If another unit transports/books:
- They must do intake documentation and supplemental notes.
- Custody transfer language is required.
