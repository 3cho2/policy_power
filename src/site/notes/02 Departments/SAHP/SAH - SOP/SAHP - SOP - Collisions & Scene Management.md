# SAHP - SOP - Collisions & Scene Management

## 1) Purpose
Standardize SAHP collision response while maintaining safe, realistic RP.

## 2) Scene command
- First On Scene is IC until relieved.
- Transfer IC to jurisdictional first-arrival when applicable.

## 3) Minimum scene actions
- Establish comms plan (primary/TAC)
- Secure hazards and manage traffic flow
- Identify medical needs and request EMS if applicable
- Document key facts if collision is significant

## 4) Documentation
- Significant collisions should be documented via report.
- Use `PP-YYYYMMDD-SAHP-####`.
